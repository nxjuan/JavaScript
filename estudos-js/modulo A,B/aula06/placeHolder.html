PlaceHolder

O que é placeholder?

PlaceHolder é uma forma de atribuir o valor de uma variavel dentro de um texto.
Mas como isso funcion?

Suponhamos que var nome = Juan.

Médoto convencional: ('É um prazer te conhecer ' + nome)
Método Template String: (É um prazer te conhecer ${nome})

Regras:

  • Deve estar entre craze
  • O nome da variavel fica entre ${}
